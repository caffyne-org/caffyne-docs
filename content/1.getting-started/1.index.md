Caffyne Shell is a modern, GTK-based desktop shell built on top of Fabric, Python, and GTK. It features a highly customisable drag-and-drop panel, fluid animations, and deeply integrated system applets.

## Features

- **Modern UI Architecture:** Built with native GTK widgets running smoothly on modern Wayland compositors.
- **Dynamic personalising:** Powered by *Matugen* to deliver seamless *Material You* color palettes derived dynamically from your wallpapers.
- **Interactive Control Hub:** 15 pre-built applets covering everything from process management to a built-in quick settings panel.
- **Modular Bar Design:** A drag-and-drop bar structure optimized for flexibility.

---

## Supported Window Managers

Caffyne Shell does not manage window configurations but connects natively to Wayland compositors. Here is the current compatibility status:

| Window Manager | Status |
| -------------- | ------ |
| **Niri** | 🟢 Stable |
| **Hyprland** | 🧪 Beta |
| **MangoWM** | 🧪 Beta |

::note
Support for additional Wayland distributions and compositors is actively planned for future releases.
::
